# Metrics-Analysis-A-B-testing
# Project Description:

This project analyzes the results of an A/B test to determine if specific changes improved key business metrics. The study includes data processing, statistical testing, and final visualization.

# Technical Stack:

Python (Pandas, Scipy) for data analysis.

Jupyter Notebook for the workflow and calculations.

Tableau for interactive dashboards and reporting.

# Project Steps:

Data Cleaning: Removing outliers and preparing the dataset for analysis.

Statistical Analysis: Testing hypotheses using p-value, confidence intervals, and normality tests.

Visualization: Creating a dashboard in Tableau to show the difference between control and test groups.
